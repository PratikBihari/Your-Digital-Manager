# YDM - Your Digital Manager Website

A modern, responsive website for YDM (Your Digital Manager), a digital marketing agency. The website features a sleek light and blue color scheme with interactive elements and smooth animations.

## Features

- Responsive design that works on all devices
- Modern black and gold color theme
- Interactive service cards with hover effects
- Smooth scrolling navigation
- Mobile-friendly navigation menu
- Animated elements on scroll
- Social media integration
- Clean and professional layout

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Flexbox/Grid)
- JavaScript (ES6+)
- Font Awesome Icons

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required

## File Structure

```
YDM/
├── index.html          # Main HTML file
├── styles.css          # CSS styles
├── script.js           # JavaScript functionality
└── README.md          # Project documentation
```
